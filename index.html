Habilidades

Co dělám ?

HTML, CSS

HTML (Hypertext Markup Language) é a linguagem usada para estruturar o conteúdo de páginas da web, como texto, imagens e vídeos, organizando-os em elementos como títulos, parágrafos e links. Juntamente com o CSS (Cascading Style Sheets), que controla a apresentação visual da página, como layout, cores e fontes, eles formam a base do desenvolvimento web, permitindo criar páginas visualmente atraentes e bem estruturadas.

JavaScript

JavaScript é uma linguagem de programação usada principalmente para tornar páginas da web interativas. Ela manipula o conteúdo e o estilo das páginas, tornando-as dinâmicas e responsivas. É essencial para o desenvolvimento front-end e cada vez mais usado no back-end também.

Java

Java é uma linguagem de programação versátil, conhecida por sua portabilidade e segurança. É usada para desenvolver uma variedade de aplicativos, desde aplicativos de desktop até sistemas empresariais e aplicativos móveis.

Projetos

Jogo do pong

O jogo Pong da Alura é uma versão moderna e simplificada do clássico jogo de arcade Pong. Os jogadores controlam raquetes em cada lado da tela e tentam rebater uma bola de um lado para o outro, evitando que ela passe pela raquete e alcance a borda oposta da tela.

Robotron

O jogo Robotron da Alura é uma recriação do clássico jogo de arcade Robotron: 2084. Nele, os jogadores enfrentam hordas de inimigos enquanto tentam salvar humanos e sobreviver.

Midi

O jogo da Música da Alura é uma divertida experiência interativa que combina música e gameplay. Os jogadores interagem com elementos musicais na tela, como notas e batidas, para criar sua própria música enquanto jogam.
